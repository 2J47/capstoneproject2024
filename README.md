## 역할 분담
총 4명 (Backend 1, Frontend 1, Fullstack 1, AI 1)

**Backend (User API)**
 * 모바일 앱 서비스를 위한 RESTful API 서버 설계 및 구현
 * 메인 데이터베이스 모델링 및 스키마 구현

**Frontend (Mobile App)**
* 사용자용 안드로이드 애플리케이션 개발
* 카메라 연동 및 이미지 검색 기능 구현

**Fullstack (Admin web)**
* 서비스 운영을 위한 관리자 대시보드 풀스택 개발
* 회원 관리 기능 구현

**AI**
* 알약 이미지 분류 모델 연구 및 학습
* Python-Node.js 간 프로세스 통신 인터페이스 구축
